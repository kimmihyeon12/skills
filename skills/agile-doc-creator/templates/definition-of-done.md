# Definition of Done

## 개요

이 문서는 모든 User Story가 "Done" 상태로 전환되기 위해 충족해야 하는 공통 완료 기준을 정의한다.

## 코드 품질

- [ ] 코드 리뷰 완료 (최소 1명 승인)
